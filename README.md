# X (Twitter) Clone

A modern, feature-rich clone of the X (formerly Twitter) platform built with **pure HTML, CSS, and JavaScript** 

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge

## No Dependencies

This project uses **zero external libraries**. It's built with:
- Pure HTML5
- Vanilla CSS3
- Native JavaScript (ES6+)
